# 택시 팟 형성하기
이 프로그램은 **학생들과 직장인**들을 위해 **택시 비용**을 아낄 수 있게 카풀을 형성하기 위해 만들었습니다
- 먼저, 회원가입을 진행합니다: 이메일과 비밀번호를 설정한 후, 다시 한 번 로그인합니다
- 역 이름을 입력하여 가까운 곳에 택시 팟이 있는지 찾을 수 있습니다
- 동승하기를 클릭하고, 인원이 찼다면 카카오톡 오픈채팅방 링크가 뜹니다.
- 팟 만들기를 클릭하여 출발 지하철역, 도착지, 출발 시간, 그리고 최대인원을 입력하여 직접 택시 팟을 형성할 수 있습니다.

  [택시팟 형성하러 가기](https://koreayuni-crypto.github.io/antigravity-study/)
